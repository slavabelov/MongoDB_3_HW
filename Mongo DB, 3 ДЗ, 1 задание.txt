Цель практической работы:
Научиться выполнять простые запросы в Redis.

Что нужно сделать
Напишите последовательность команд для Redis:
Создайте ключ index со значением “index precalculated content”.
Проверьте, есть ли ключ index в БД.
Узнайте, сколько ещё времени будет существовать ключ index.
Отмените запланированное удаление ключа index.

Для выполнения заданных операций в Redis понадобятся следующие команды:

1. Создание ключа index со значением "index precalculated content":

SET index "index precalculated content"


2. Проверка наличия ключа index в базе данных:

EXISTS index


3. Получение оставшегося времени жизни ключа index (времени до истечения таймаута):

TTL index

4. Отмена запланированного удаления ключа index:

PERSIST index

Выполнение этих команд позволит  создать ключ index с заданным значением, проверить его наличие, узнать оставшееся время жизни и отменить запланированное удаление.
